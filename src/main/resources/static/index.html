How to call the API

Example commands (PowerShell):

Health check
Invoke-RestMethod http://localhost:8080/api/health
List users
Invoke-RestMethod http://localhost:8080/api/user
List restaurants
Invoke-RestMethod http://localhost:8080/api/restaurant

Note: endpoints like /api/order/id/1 require data to exist in the database.
